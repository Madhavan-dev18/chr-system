# **App Name**: MediSafe

## Core Features:

- Registration & Login: Users can register with their basic information to create a unique profile and log in securely to access their health data.
- Health ID Generation: Automatically generate a unique, 10-digit Health ID upon user registration.
- Health Data Entry: Allow users to input and update their personal and medical information, including personal details, medical history, and lifestyle data. Focus on entering health data and ignore any ability to upload files, because files will bloat the data layer
- Profile Viewing: Users can view their complete health profile, including all entered data, in a consolidated view.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which translates to a shade of blue (#3399CC), conveying trust and security.
- Background color: HSL(210, 20%, 95%) which translates to a very light blue (#F0F8FF), providing a clean and calm backdrop.
- Accent color: HSL(180, 60%, 40%) which translates to a turquoise (#33CCCC), for interactive elements, providing a sense of clarity and well-being.
- Font pairing: 'Inter' (sans-serif) for body text to ensure readability, paired with 'Playfair' (serif) for headlines, lending an elegant feel to key information.
- Use a consistent set of simple, line-based icons to represent different data categories, enhancing usability.
- Employ a clean, card-based layout with ample white space to present information in an organized and digestible manner.
- Incorporate subtle transitions and animations for user interactions, such as loading or data updates, to improve user experience.
